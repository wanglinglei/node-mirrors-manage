### npm 镜像源管理工具

#### 安装

```
npm install -g mnpmmior
```

#### 查看帮助

```
mnpmmior help
```

#### 查看镜像源列表

```
mnpmmior list
```

#### 查看当前镜像源

```
mnpmmior current
```

#### 切换镜像源

```
mnpmmior use
```

#### 添加镜像源

```
mnpmmior add
```

#### 删除镜像源

```
mnpmmior del
```

#### 测试镜像源连接

```
mnpmmior ping
```
